+ Le fichier principal à exécuter est Pygraphe.py
+ Pour charger ou sauvegarder un graphe, il faut indiquer un nom de fichier simple, sans extension. Toutes les sauvegardes se trouvent à la racine du dossier Save (pas d’autre sous-dossier possible).
+ Avec une version ancienne de python, il peut être nécessaire de créer un fichier (vide) dans le dossier Save nommé __init__.py
